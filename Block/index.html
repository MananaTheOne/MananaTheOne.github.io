What is "BLOCK" and why "BLOCK"?

"BLOCK" is a Discord API that has Auto-Moderation and Manual-Moderation (Commands),
both differ from each other. The idea for block started when I wanted to-
create a moderation bot for my server with a few rules. I wanted it so that-
it is easily accessable and user friendly.

Commands

- /help [command | none]
- /kick [member]
- /mute [member]
- /ban [member]
- /tempban [seconds] [member]
- /tempmute [seconds] [member]
- /purge [amount]
- /unban [member]
- /unmute [member]

Features

- Easily Accessable Debug Commands
- Uses Slash Commands (instead of the ol' ?ping command)
- Better Auto-Moderation
- Anonymous Mode (hidden command output)
- Untrusted by people, and I dont know the fuck why
